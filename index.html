Human Computer Interaction Project

Coursework project for CPT208. Our system demonstrates user-centered interface design principles and focuses on intuitive interaction, visual feedback and usability.

Project Introduction

This project explores how human-computer interaction principles can improve digital system usability. The system aims to create a user-friendly interface that allows efficient interaction between users and software systems.


Key objectives include:

  • Design intuitive user interfaces
  • Improve interaction efficiency
  • Provide responsive feedback
  • Evaluate usability and user experience
